# OpenHarmony_OHScrcpy 安装说明
本文档详细介绍各平台（Windows/Linux/macOS）的安装说明。


## 系统要求
- **操作系统**：Windows/Linux/macOS
- **Python版本**：Python 3.7或更高版本

## Windows平台安装方法
  特别注意：安装前请暂时关闭杀毒软件。
  1. 双击运行`OHScrcpy_Setup_{版本号}.exe`，弹出**OpenHarmony OHScrcpy安装向导**界面；
  2. 点击`下一步`，出现**安全告警说明**页面，点击`勾选`项确认安全说明；
  3. 点击`下一步`，出现**选择安装目录**页面，选择安装目录（可不选，使用默认安装目录）；
  4. 点击`下一步`，出现**选择开始菜单文件夹**页面，保持默认即可；
  5. 点击`下一步`，出现**选择附加任务**页面，保持默认即可，也可以根据自身实际需要选择；
  6. 点击`下一步`，出现**准备安装**页面，确认安装信息；
  7. 点击`安装`，开始自动安装；
  8. 点击`完成`，完成安装任务，后续可从**桌面快捷方式**或者**安装目录下**双击`OHScrcpy.exe`启动投屏工具。

## Linux/macOS平台安装方法
  1. 解压安装包，执行命令：`unzip OHScrcpy_{平台名}_{版本号}.zip -d {解压目录} -q`；
  2. 进入{解压目录}，执行安装脚本`./install_ohscrcpy.sh`，根据提示**选择安装目录**，完成安装；
  3. 根据提示，完成环境变量重新导入，`source $HOME/.bashrc`（仅供参考，根据实际提示内容操作）；
  4. 任意目录执行`OHScrcpy`/`ohscrcpy`均可直接启动投屏工具；
  5. 任意目录执行`ohscrcpy-uninstall`，可以自动完成卸载投屏工具。
