===================================================================
                 OpenHarmony OHScrcpy 安装前说明
===================================================================

感谢您选择OpenHarmony OHScrcpy！在安装前，请仔细阅读以下重要信息。

一、关于 OHScrcpy
------------------------------------------------
OHScrcpy 是一个开源的 OpenHarmony 系统屏幕镜像工具，功能类似于Android平台的scrcpy镜像工具，提供了：
• 更好的图形用户界面
• 一键连接功能
• 屏幕录制
• 键盘和鼠标映射
• 多设备同时连接

源码仓库：https://gitcode.com/luodh0157/OpenHarmony_Scrcpy
------------------------------------------------

二、系统要求
------------------------------------------------
[最低配置]
• 操作系统：Windows 7/8/10/11 (64位)
• CPU：Intel Core i3 或同等性能
• 内存：4GB RAM
• 存储空间：200MB 可用空间
• .NET Framework 4.6.1 或更高版本

[推荐配置]
• 操作系统：Windows 10/11 (64位)
• CPU：Intel Core i5 或更好
• 内存：8GB RAM 或更多
• 存储空间：500MB 可用空间
• .NET Framework 4.8
------------------------------------------------

三、OpenHarmony设备要求
------------------------------------------------
1. OpenHarmony5.0或更高版本
2. 已启用开发者选项
3. 已启用USB调试模式
4. 使用原装或高质量USB数据线
------------------------------------------------

四、重要安全说明（请务必阅读！）
------------------------------------------------
由于OpenHarmony OHScrcpy 是开源免费软件，我们未购买昂贵的代码签名证书，
Windows可能会显示安全警告。这是正常现象，不是病毒！

安装时可能遇到的安全警告及解决方法：

情况一：SmartScreen 筛选器警告
   显示："Windows 已保护你的电脑"
   操作：点击"更多信息" → 点击"仍要运行"

情况二：用户账户控制警告
   显示："你要允许此应用对你的设备进行更改吗？"
   操作：点击"是"

情况三：发布者未知警告
   显示："发布者：未知"
   操作：点击"运行"或"是"

为什么会出现这些警告？
1. 代码签名证书价格昂贵
2. 开源项目通常没有资金购买证书
3. Windows要求所有软件必须签名

如何验证软件安全性？
1. 验证文件哈希值（见安装目录下的hash.txt）
2. 查看开源代码：https://gitcode.com/luodh0157/OpenHarmony_Scrcpy

五、安装步骤
------------------------------------------------
1. 下载安装程序 OHScrcpy_Setup_v1.0.0.exe
2. 双击运行安装程序
3. 如果出现安全警告，按上述方法处理
4. 按照安装向导的提示操作
5. 选择安装目录（建议使用默认目录）
6. 选择需要创建的快捷方式
7. 等待安装完成
------------------------------------------------

六、安装前准备工作
------------------------------------------------
1. 关闭所有杀毒软件（临时关闭，安装后重新开启）
2. 确保OpenHarmony设备已连接电脑
3. 在OpenHarmony设备上启用USB调试：
   a) 进入"设置" → "关于手机"
   b) 连续点击"版本号"，开启开发者模式
   c) 返回"设置" → "开发者选项"
   d) 开启"USB调试"

4. 如果是首次连接，需要在设备上允许USB调试
------------------------------------------------

七、故障排除（安装前）
------------------------------------------------
如果无法安装，请尝试以下方法：

1. 以管理员身份运行安装程序
   右键点击安装程序 → "以管理员身份运行"

2. 关闭Windows Defender实时保护（临时）
   • 打开"Windows安全中心"
   • 点击"病毒和威胁防护"
   • 点击"管理设置"
   • 关闭"实时保护"

3. 添加安装程序到信任列表
   • 右键点击安装程序 → "属性"
   • 勾选"解除锁定"（如果有）

4. 检查系统兼容性
   • 确保系统是64位Windows
   • 安装最新的Windows更新
   • 安装.NET Framework更新
