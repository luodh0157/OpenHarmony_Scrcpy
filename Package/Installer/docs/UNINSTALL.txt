===================================================================
                  OpenHarmony OHscrcpy 卸载说明
===================================================================

如果您需要卸载OpenHarmony OHscrcpy，请按照以下步骤操作：

一、推荐卸载方法
------------------------------------------------
1. 打开"控制面板" → "程序" → "程序和功能"
2. 在程序列表中找到"OpenHarmony OHscrcpy"
3. 右键点击并选择"卸载"
4. 按照卸载向导完成操作

或者：

1. 点击开始菜单，找到"OpenHarmony OHscrcpy"文件夹
2. 点击"卸载 OpenHarmony OHscrcpy"
3. 按照卸载向导完成操作
------------------------------------------------

二、卸载选项说明
------------------------------------------------
卸载过程中，您会看到以下选项：

1. 是否保留个人数据？
   • 选择"是"：保留 {app}\Data 和 {app}\Config 目录
   • 选择"否"：删除所有用户数据
   
   建议：如果您计划重新安装，可以选择保留数据。

2. 是否删除配置文件？
   • 选择"是"：删除所有配置文件
   • 选择"否"：保留配置文件
   
   建议：如果您遇到问题需要重新安装，建议删除配置文件。
------------------------------------------------

三、完全卸载（手动步骤）
------------------------------------------------
如果通过正常方式无法卸载，可以手动完全卸载：

1. 停止运行中的程序：
   • 按 Ctrl+Shift+Esc 打开任务管理器
   • 结束所有运行中的OHscrcpy.exe进程

2. 删除安装目录：
   • 默认位置：C:\Program Files\OpenHarmony OHscrcpy
   • 如果自定义了安装位置，请到相应目录删除

3. 删除用户数据：
   • 打开资源管理器，输入：%APPDATA%
   • 删除OpenHarmony OHscrcpy 文件夹（如果存在）
   
   • 打开资源管理器，输入：%LOCALAPPDATA%
   • 删除OpenHarmony OHscrcpy 文件夹（如果存在）

4. 删除注册表项（高级用户）：
   警告：修改注册表有风险，请谨慎操作！
   • 按 Win+R，输入 regedit，回车
   • 删除以下键值（如果存在）：
     HKEY_CURRENT_USER\Software\OpenHarmony OHscrcpy
     HKEY_LOCAL_MACHINE\SOFTWARE\OpenHarmony OHscrcpy
   
   • 从PATH环境变量中移除OpenHarmony OHscrcpy路径：
     HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment
     编辑Path值，删除OpenHarmony OHscrcpy的路径

5. 删除快捷方式：
   • 桌面上的OpenHarmony OHscrcpy快捷方式
   • 开始菜单中的OpenHarmony OHscrcpy文件夹
------------------------------------------------

四、卸载后清理
------------------------------------------------
卸载完成后，建议：

1. 重启计算机（确保所有文件被释放）
2. 运行磁盘清理工具
3. 检查是否还有残留文件
------------------------------------------------

五、重新安装
------------------------------------------------
如果您卸载后想重新安装：

1. 确保完全卸载旧版本
2. 从官方网站下载最新版本
3. 按照安装说明重新安装
------------------------------------------------

六、反馈卸载问题
------------------------------------------------
如果在卸载过程中遇到问题：

1. 查看详细日志：
   • 卸载程序会在桌面生成 ohscrcpy_uninstall.log
   • 打开查看具体错误信息

2. 获取帮助：
   • GitCode Issues：https://gitcode.com/luodh0157/OpenHarmony_Scrcpy/issues
   
   反馈时请提供：
   • 操作系统版本
   • 错误截图
   • ohscrcpy_uninstall.log 内容
