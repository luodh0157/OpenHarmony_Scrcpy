# OpenHarmony_Scrcpy H.265支持与分辨率校验设计方案

> 制定日期：2026-04-22
> 更新日期：2026-04-25
> 状态：已实现

---

## 一、概述

本文档包含两个功能设计方案：
1. **H.265视频流支持** - 新增HEVC编解码能力
2. **分辨率校验与降级** - 确保编解码器支持目标分辨率

---

## 二、分辨率与Codec选择策略（核心设计）

### 2.1 选择流程

```
服务启动
    │
    ▼
getPrimaryScreenInfo() → 获取原始分辨率
    │
    ▼
Step 1: checkHevcEncoderExists()
    │       检测HEVC硬件编码器是否存在
    │
    ├─→ 不存在 ─────────────────────────────┐
    │                                        │
    └─→ 存在                                 │
            │                                │
            ▼                                │
    Step 2: checkHevcSizeAndFrameRateSupported()
            检测HEVC是否支持原始分辨率        │
            │                                │
            ├─→ 支持 ──→ 使用HEVC + 原始分辨率 ─┤
            │                                │
            └─→ 不支持                        │
                    │                        │
                    ▼                        │
            Step 3: findCodecSupportedResolution("h265")
            从HEVC标准列表找支持的分辨率      │
                    │                        │
                    ├─→ 找到 ─→ 使用HEVC + 该分辨率
                    │                        │
                    └─→ 未找到 ───────────────┤
                                             │
                                             ▼
                                Step 4: findCodecSupportedResolution("h264")
                                从H264标准列表找支持的分辨率
                                             │
                                             ├─→ 找到 ─→ 使用H264 + 该分辨率
                                             │
                                             └─→ 未找到 ─→ 强制720x1280（警告）
```

### 2.2 核心函数

| 函数名 | 职责 |
|--------|------|
| `checkHevcEncoderExists()` | 仅检测HEVC硬件编码器是否存在（不检测分辨率） |
| `checkHevcSizeAndFrameRateSupported()` | 检测HEVC是否支持特定分辨率+帧率 |
| `checkAvcSizeAndFrameRateSupported()` | 检测AVC是否支持特定分辨率+帧率 |
| `findCodecSupportedResolution()` | 从指定codec的标准列表找支持的分辨率 |

### 2.3 标准分辨率列表

**横屏（16:9）**：
- 4K (3840x2160), 2K (2560x1440), 1080p (1920x1080), 720p (1280x720), 480p (854x480), 360p (640x360), 240p (426x240)

**竖屏（9:16）**：
- 4K (2160x3840), 2K (1440x2560), 1080p (1080x1920), 720p (720x1280), 480p (480x854), 360p (360x640), 240p (240x426)

## 三、命名规范

| 位置 | 规范 | 示例 |
|-----|------|-----|
| 函数名 | 使用 `avc`/`hevc` | `checkHevcEncoderExists()`, `printHevcVideoCodecCapability()` |
| 变量名 | 使用 `avc`/`hevc` | `hasHevc`, `hevcRes`, `avcRes` |
| 打印信息 | 使用 `H.264`/`H.265` | "H.265 hardware encoder found"（便于用户理解） |
| 配置字符串 | 使用 `h264`/`h265` | `screenInfo.codec = "h265"` |
